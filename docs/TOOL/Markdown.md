## 基本语法  

`#`: $~~$标题创建  
需要在`#`后空一格来构成标题，对于`#`为一级标题，`##`为二级标题，以此类推，想要分目录必须为一级标题 + 多级标题结构，切忌出现**多个一级标题**  

`**`: $~~$对文字加粗，需要在要加粗的文字前后加上即可  

`*`: $~~$对文字倾斜处理，前后加上  

`***`: $~~$对文字进行倾斜和加粗处理  

`<space><space>`和`\n`: $~~$段落换行处理，要么在段的后尾加上两个空格，或者直接单独空一行  

`` ` ` ``: $~~$插入一段代码段，使用``包围代码   

`` ``` ``` ``: $~~$插入代码块，用如下结构,`language`为你要的语言  
``` Markdown

    ``` language
        code here
    ```
     
```  
$~~$  
`$`: $~~$插入数学LATEX表达式  

`>`: $~~$引用块，在开头段落加上后空格  

`-`: $~~$无序列表，该列表没有顺序之分  

`<number>. `: $~~$有序列表，从1开始，有顺序，多级嵌套内部会是abc...  

`[link_name](link_addr "heading")`: $~~$创建链接，其中`link_name`和`link_addr`为必填，而`heading`为选填  

`![png_name](png_addr)`: $~~$插入图片，`png_name`可以任意，一般都不会显示

## 排版技巧  
1. $~~$空多行，使用`html`标签--`<br><\br>`；要想自定义行数一般结合latex表达式的空格表达式单独占一行，再正常空行即可，即`$~~$`后在`<space><space>`  

2. $~~$段落开头空格，使用latex的`$~~~~$`即可，代表空四个空格  

3. $~~$嵌套使用注意**缩进**对排版的影响
